Solicitation Search

Search all federal solicitations on SAM.gov — by keyword, by NAICS code, or both. Choose open, closed, or both; results are ordered by soonest response deadline first. No fixed topics or editions. Requires a free SAM.gov API key.

Tip: SAM.gov matches the keyword against the notice title. Combine with a NAICS code to narrow by industry. Get a free key at sam.gov → Workspace → Account Details → API Key; it is stored only in this browser and sent only to SAM.gov. On the hosted site, api.sam.gov may block datacenter IPs — the local version always works.
